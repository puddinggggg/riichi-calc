# Riichi Calc Web

React + styled-components 기반 리치마작 웹앱입니다.

## 반영 내용

- 초기 화면 메뉴 분리
  - 대기패 확인
  - 점수계산기
- 대기패 확인 화면
  - 13장 패 선택
  - 완성 가능한 대기패 목록 표시
  - 선택된 패/선택할 패 버튼에 마작패 유니코드 크게 표시
- 점수계산기 화면
  - 역 선택 기반 점수 계산
  - 도라/우라도라/적도라를 표시패 선택이 아니라 개수 입력 방식으로 변경
  - 리치봉 항목 삭제
  - 멘젠쯔모 선택 시 계산 조건의 화료 방식도 쯔모로 자동 변경
- 검증 테스트 UI는 표시하지 않도록 제외

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
